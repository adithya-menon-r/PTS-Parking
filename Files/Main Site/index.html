Preface

Our project helps in the proper mamagement of parking by making the system much more simple, reliable and secure in places where parking is a hassle. By making use of File Handling and adding some very handy and necessary features, our project, “ A Parking Management System”, can check for the availablity of parking spots whithin fractions of seconds, collect information from the user, assign the parking spot and record the entry log.When the user wants to exit the parking spot, it can reverse the process, log th exit, calculate the amount to be paid based on a rate and also provide the option of UPI as a form of payment. It also implements admin functions and much more.

Libraries / Modules used

Pickle

Pickle is an inbuilt Python module that is primarily used in serializing and deserializing a Python object structure. Here, it is used to enable file handling capabilities - to log an entry / exit in a .dat or .bin file and to read it when necessary.

PyAutoGUI

PyAutoGUI lets your Python scripts control the mouse and keyboard to automate interactions with other applications. It also helps implement basic GUI elements like a secure password input box. PyAutoGUI works on Windows, macOS, and Linux, and runs on Python 2 and 3.

PyGame

PyGame is commonly used for the development of multimedia applications like video games using Python. It uses the Simple DirectMedia Layer library and several other popular libraries. Here, pygame is used to display the QR Code (a image) in a seperate window.

  • We can also create a mobile app that can be used by regular users. This can also bring the added functionality of subscription service where in the user pays a fixed amount every month to park along with some benefits. This can significantly improve the financial aspect of such a business.

  • To avoid inconveniences due to poor parking, we can add parallel systems in each parking spot, that based on sensor and camera input determines whether the user has parked the vehicle correctly. If not, it can flash a red LED informing the user that it may cause inconvenience to other users.
